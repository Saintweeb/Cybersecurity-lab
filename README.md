# Cybersecurity Labs

This repository documents my hands-on learning in cybersecurity, including offensive security, defensive operations, and digital forensics. It contains notes, lab writeups, and lessons learned from practical exercises on TryHackMe.

## Structure

- **Offensive/** – Labs focused on finding vulnerabilities and ethical exploitation (red team skills).  
- **Defensive/** – Labs focused on SOC monitoring, incident response, and threat detection (blue team skills).  
- **Forensics/** – Labs focused on digital forensics and threat intelligence exercises.

## Skills Highlighted

- Offensive security: Vulnerability identification, ethical exploitation, web app security.  
- Defensive security: Log analysis, SOC monitoring, incident response workflows.  
- Digital forensics & threat intelligence: Evidence collection, analysis, and threat research.

## Note

This repository **does not contain any real flags, paid content, or sensitive data**. All work reflects ethical learning and personal notes.
